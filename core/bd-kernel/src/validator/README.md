# Overview

The Contract Validator verifies that every `contract.json` file in the BD Universe platform follows the BDOS Universal Contract Specification.

Validation ensures that all engineering artifacts expose consistent metadata and can safely participate in platform automation.

---

# Responsibilities

- Validate contract structure
- Verify required fields
- Report validation errors
- Support future schema versions
