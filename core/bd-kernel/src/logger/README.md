# Overview

The Logger is the centralized logging service for the BD Universe platform.

Every platform service writes messages through the Logger instead of interacting directly with the browser console.

The Logger provides a consistent history of platform activity and serves as the foundation for diagnostics, monitoring, and future developer tools.

---

# Responsibilities

- Record information messages
- Record warnings
- Record errors
- Maintain log history
- Provide log retrieval
