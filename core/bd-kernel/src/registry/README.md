# Overview

The Service Registry is the central directory of the BD Kernel.

It stores shared platform services and provides controlled access to them throughout the BD Universe ecosystem.

The Registry eliminates direct dependencies between services by providing a single access point.

---

# Responsibilities

- Register services
- Retrieve services
- Check service availability
- Prevent duplicate registrations
- List registered services

---

# Consumers

- Repository Service
- Explorer
- Dashboard
- Composer
- Workspace
