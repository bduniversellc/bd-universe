# Overview

The Event Bus is the communication layer of the BD Kernel.

It enables platform services to exchange information without direct dependencies.

Every service can publish events and subscribe to events generated by other services.

---

# Responsibilities

- Publish events
- Subscribe to events
- Remove subscriptions
- Notify listeners
- Decouple platform services

---

# Consumers

- Repository Service
- Explorer
- Dashboard
- Composer
- Workspace
- Logger

---

# Guiding Principle

Services communicate through events, not direct references.
