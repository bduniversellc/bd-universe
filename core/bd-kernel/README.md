# Overview

BD Kernel is the core runtime of the BD Universe platform.

It provides the shared services used by every application, package, and engineering tool.

Rather than allowing platform services to communicate directly with one another, the Kernel coordinates communication through a unified API.

BD Kernel is the foundation of BDOS.
