# BD Design System Specification

ID: BDDS

Version: 1.0.0

Status: Production

---

## Objective

Create a consistent visual language for every BD Universe product.

---

## Scope

Typography

Color

Spacing

Grid

Motion

Components

Accessibility

---

## Rules

Never hardcode colors.

Never hardcode spacing.

Never hardcode typography.

Always use Design Tokens.

Always build reusable components.

---

## Success Criteria

Every application should look like part of the same ecosystem while preserving its own identity.
