# BD Universe Design Tokens

Version: 1.0.0

## Purpose

Design Tokens are the single source of truth for every visual value used throughout the BD Universe ecosystem.

Every website, application, component, dashboard and future product references these tokens.

## Principles

- Never hardcode colors.
- Never hardcode spacing.
- Never hardcode typography.
- Everything references tokens.

## Categories

- Colors
- Typography
- Spacing
- Radius
- Shadows
- Motion
- Layout
- Breakpoints

## Future

These tokens will later synchronize with:

- BDUI
- Figma
- AI Agents
- Future mobile applications
