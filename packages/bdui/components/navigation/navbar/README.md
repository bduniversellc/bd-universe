# BDUI-NAV-001 Navbar

Version: 1.0.0

Status: Draft

---

## Purpose

The Navbar is the primary navigation component for all BD Universe applications.

It provides users with a consistent, accessible, and responsive navigation experience across desktop and mobile devices.

---

## Features

- Semantic HTML5
- Responsive
- Sticky Navigation
- Mobile Menu
- Keyboard Accessible
- Theme Ready
- Dark Mode Ready
- Lightweight Vanilla JavaScript
