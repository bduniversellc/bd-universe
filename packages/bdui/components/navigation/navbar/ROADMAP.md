# Vision

The Navbar is the primary navigation component for all BD Universe applications.

It provides a consistent, accessible, and responsive navigation experience while serving as the foundation for every future website and application built on BDUI.

---

# Mission

Build a lightweight, reusable navigation component that:

- is semantic
- is accessible
- is responsive
- works without frameworks
- integrates seamlessly with BDDS
