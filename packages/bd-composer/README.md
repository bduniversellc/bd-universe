# Registry

Package: BD Composer

Version: 2.0.0

---

## Purpose

The Registry is the central catalog of BD Universe.

It stores every reusable component, blueprint, template and future platform service.

Composer never searches the repository directly.

Composer asks the Registry.

---

## Responsibilities

- Register components
- Register Blueprints
- Register Templates
- Lookup components
- Return registered objects
