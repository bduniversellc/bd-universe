# Overview

BD Composer is the composition engine of the BD Universe platform.

It assembles applications from reusable Blueprints and BDUI components while separating business logic from presentation.

Composer is a reusable platform package shared by every application in the BD Universe ecosystem.

---

# Responsibilities

- Initialize applications
- Load configuration
- Register components
- Resolve routes
- Render pages
- Manage application composition
