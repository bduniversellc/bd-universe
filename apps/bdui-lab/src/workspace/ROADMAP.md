# BD Studio Workspace

Application: BDUI Lab

Display Name: BD Studio

Version: 1.0.0

# Purpose

The Workspace is the primary user interface of BD Studio.

Every page is displayed inside the Workspace.

The Workspace provides a consistent developer experience across every tool in BD Studio.

---

# Regions

- Header
- Sidebar
- Main Content
- Footer

---

The Workspace is intentionally independent of any specific component library.

Its responsibility is to host tools, not implement them.
