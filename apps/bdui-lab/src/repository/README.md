# Overview

The Repository Service provides a centralized source of engineering metadata for BD Studio.

Its purpose is to expose information about applications, packages, components, blueprints, workspaces, standards, and future platform assets.

The Repository Service separates data discovery from presentation.

The Explorer, Composer, and future developer tools consume this service instead of directly reading repository folders.

---

# Responsibilities

- Discover engineering assets
- Organize repository metadata
- Provide structured data
- Support automatic navigation
- Enable repository search
- Supply metadata to developer tools
