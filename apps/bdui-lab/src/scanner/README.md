Package: BD Studio

Version: 1.0.0

# Overview

The Repository Scanner discovers every engineering artifact inside the BD Universe repository.

It reads contract.json files and generates a unified catalog for developer tools.

The scanner eliminates manual maintenance of repository metadata.

---

# Responsibilities

- Scan repository
- Read contracts
- Validate contracts
- Build catalog
- Generate manifest

---

# Consumers

- Explorer
- Dashboard
- Repository Service
- Composer
