# Overview

The Dashboard is the home screen of BD Studio.

It provides a high-level overview of the entire BD Universe ecosystem, presenting repository statistics, recent activity, and quick access to engineering tools.

The Dashboard is the first screen developers see when launching BD Studio.

---

# Responsibilities

- Display repository overview
- Show engineering metrics
- Provide quick navigation
- Highlight recent changes
- Display platform health
